### Styling

**Traditional Version:**
- Main styles: `traditional/css/style.css`
- Responsive: `traditional/css/responsive.css`
- Animations: `traditional/css/animations.css`

**React Version:**
- Theme: `react-version/src/theme.js`
- Component styles: Use Material-UI `sx` prop or styled components
